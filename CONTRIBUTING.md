# WebEdit-r'a Katkıda Bulunma

Öncelikle, WebEdit-r'a katkıda bulunmayı düşündüğünüz için teşekkür ederiz! 🎉

## Davranış Kuralları

Saygılı, kapsayıcı ve yapıcı olun.

## Nasıl Katkıda Bulunabilirim?

### 🐛 Hata Bildirme

**Göndermeden önce:**
- Hatanın daha önce bildirilip bildirilmediğini kontrol edin
- Bilgi toplayın: İşletim sistemi, tarayıcı, yeniden üretme adımları

**GitHub Issues üzerinden şunlarla gönderin:**
- Açık, tanımlayıcı başlık
- Yeniden üretme adımları
- Beklenen ve gerçekleşen davranış
- Varsa ekran görüntüleri

### 💡 Özellik Önerme

GitHub Issues üzerinden özellik isteklerini şunlarla gönderin:
- Açık kullanım senaryosu
- Bu özelliğin neden yararlı olacağı
- Olası uygulama yaklaşımı

### 🔧 Pull Request'ler

1. **Depoyu fork edin**
2. **Bir branch oluşturun:** `git checkout -b feature/benim-ozelligim`
3. **Değişiklik yapın:**
   - Mevcut kod stilini takip edin
   - Karmaşık mantık için yorum ekleyin
   - Değişikliklerinizi test edin
4. **Commit edin:** `git commit -m "Eklendi: Benim özelliğim"`
5. **Push edin:** `git push origin feature/benim-ozelligim`
6. **PR açın** ve şunları ekleyin:
   - Değişikliklerin açıklaması
   - Varsa ilgili issue numarası
   - UI değişiklikleri için ekran görüntüleri
